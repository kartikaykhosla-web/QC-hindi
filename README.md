# QC Hindi

Streamlit app for Hindi editorial QC using Gemini on Vertex AI.

## Main app

- `qc_code_hindi.py`

## Supporting files

- `hindi_qc_rules.txt`
- `requirements.txt`
- `packages.txt`

## Local run

```bash
streamlit run qc_code_hindi.py
```

## Required secret

Add `GCP_SERVICE_ACCOUNT_JSON_B64` in Streamlit secrets.
